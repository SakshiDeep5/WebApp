A scalable web app with **authentication** and a **dashboard** (tasks CRUD), built with Next.js (frontend) and Node.js/Express (backend), as per the assignment requirements.

## Features

- **Frontend (Next.js + Tailwind)**
  - Responsive UI (TailwindCSS)
  - Forms with client-side validation (Zod + React Hook Form)
  - Protected routes (dashboard requires login)
  - Login, Register, Dashboard, Profile, Tasks CRUD, Search & filter, Logout

- **Backend (Node.js + Express)**
  - JWT-based auth (signup / login)
  - Profile get/update APIs
  - Tasks CRUD APIs with search and status filter
  - MongoDB (Mongoose), bcrypt password hashing, express-validator, auth middleware, central error handling
